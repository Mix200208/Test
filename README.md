# VoteApp

Android приложение для создания опросов. Данное приложение является курсовой работой.

## Использованные технологии

<ul>
  <li> В качестве сервера наша группа арендовала облачный сервер на Ubuntu. </li>
  <li> Приложение было написано на  Kotlin. </li>
  <li> 2 человека отвечало за сервер, 2 за создание мобильного приложения, 1 за дизайн.</li>
  
</ul>
